# 🍕 Mayar Pizza Website

A modern and visually appealing pizza restaurant website built using **HTML and CSS**.  
This project showcases a responsive layout, animations, and a complete restaurant landing page design.

## 🌟 Features

- 🏠 Navigation bar with dropdown menus (Pages, Shops, News)
- 🍕 Hero section with animated elements
- ⭐ Best seller products section
- 📋 Full menu (Pizza & Drinks)
- 🚚 Delivery section with animation
- 📞 Footer with contact details
- ✨ CSS animations and hover effects
- 🎨 Stylish and modern UI design

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Font Awesome (icons)
